**Общий DoD (применяется к каждому Stage поверх частного):**
- Код компилируется, проходит `go vet` и линтер (`golangci-lint`) / для фронта — typecheck + ESLint.

- Обновлены релевантные SPMS-файлы (`STATUS.yaml`, при необходимости `ARCHITECTURE.md`).
- Нет незакрытых блокеров по этому этапу.
